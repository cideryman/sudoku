# 스도쿠

브라우저에서 바로 실행할 수 있는 스도쿠 생성기이자 플레이 앱입니다.  
퍼즐을 직접 풀 수도 있고, A4 용지에 인쇄하거나 PDF로 저장할 수도 있습니다.

## 주요 기능

- 스도쿠 퍼즐 랜덤 생성
- 난이도 선택
  - 이지
  - 미디움
  - 디피컬트
  - 베리 디피컬트
- 유일한 해를 가지는 퍼즐 생성 검증
- 한 페이지에 1~4개 퍼즐 배치
- A4 인쇄 및 PDF 저장
- 정답 페이지 포함 옵션
- 웹페이지에서 직접 플레이
- 모바일 숫자패드
- 후보 숫자 메모 기능
- 힌트 1칸 채우기
- 풀이 검사
- 키보드 조작
- 진행 상황 자동 저장

## 실행 방법

설치 과정이 필요 없습니다.

1. `index.html` 파일을 브라우저에서 엽니다.
2. 난이도와 페이지 설정을 선택합니다.
3. `새 퍼즐 생성` 버튼을 누릅니다.
4. 화면에서 바로 풀거나 `인쇄 / PDF 저장` 버튼으로 출력합니다.

Windows PC에서는 `Sudoku.exe`를 실행해도 됩니다.  
단, `Sudoku.exe`와 `index.html`은 같은 폴더에 있어야 합니다.

## 플레이 방법

빈칸을 선택한 뒤 숫자패드에서 숫자를 누르면 입력됩니다.

- `메모` 버튼: 후보 숫자 입력 모드로 전환
- `일반` 버튼: 일반 숫자 입력 모드로 전환
- 메모 모드에서 같은 숫자를 다시 누르면 해당 후보 숫자만 삭제
- `지움`: 선택한 칸 삭제
- `풀이 검사`: 입력한 숫자가 맞는지 확인
- `힌트 1칸`: 빈칸 하나를 정답으로 채우고 노란색으로 표시
- `정답 보기`: 전체 정답 표시

## 키보드 조작

- 방향키: 빈칸 이동
- `1`~`9`: 숫자 입력
- `Backspace`, `Delete`, `0`: 선택한 칸 지우기
- `M`: 메모/일반 모드 전환

## 인쇄 및 PDF 저장

`인쇄 / PDF 저장` 버튼을 누르면 브라우저 인쇄 창이 열립니다.

- 실제 프린터를 선택하면 A4 용지로 인쇄할 수 있습니다.
- `PDF로 저장`을 선택하면 PDF 파일로 저장할 수 있습니다.
- `정답 페이지 포함`을 켜면 문제 뒤에 정답 페이지가 함께 출력됩니다.

## 모바일 사용

아이패드나 스마트폰에서도 브라우저로 실행할 수 있습니다.

`index.html` 파일을 iCloud Drive, Google Drive, OneDrive, 카카오톡, 이메일 등으로 옮긴 뒤 모바일 브라우저에서 열면 됩니다. GitHub Pages로 배포하면 링크만으로 접속할 수 있습니다.

화면 폭이 768px 이하인 기기에서는 플레이 모드만 크게 표시되며, 인쇄용 A4 미리보기와 인쇄 버튼은 숨겨집니다. 모바일에서는 한 번에 퍼즐 1개만 생성합니다.

## PWA 설치

GitHub Pages처럼 HTTPS로 배포하면 PWA로 설치할 수 있습니다.

- Android Chrome: 메뉴에서 `홈 화면에 추가` 또는 `앱 설치`
- iPhone/iPad Safari: 공유 버튼에서 `홈 화면에 추가`
- PC Chrome/Edge: 주소창의 설치 아이콘 또는 메뉴의 `앱 설치`

서비스 워커는 `file://`로 직접 열 때는 등록되지 않고, HTTPS 또는 localhost 환경에서 동작합니다.

## 자동 저장

플레이 중인 퍼즐, 입력한 숫자, 메모, 힌트, 설정은 브라우저의 `localStorage`에 자동 저장됩니다.  
새로고침해도 이전 진행 상태를 이어서 플레이할 수 있습니다.
